# GorbageTruck.fun

Gorbage Truck Rush is a React + Vite game for gorbage gobblers everywhere.

## Run locally

Prerequisites: Node.js 18+

1) Install dependencies: `npm install`
2) Start the dev server: `npm run dev` (default at http://localhost:5173)

## Build

`npm run build`

## Deploy

Builds and publishes to GitHub Pages: `npm run deploy`
